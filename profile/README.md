# Simple Blog
## Пет-проект личного блога
Я Иван Смирнов, Frontend-разработчик и Linux Enjoyer. Данная организация - пет-проект своего блога по проге, который разрабатывается в образовательных целях.
## Технологический стек
Проект состоит из клиента и сервера:
### Frontend
- TypeScript
- pnpm
- Preact + Router

### Backend
- Rust
- Diesel.rs (ORM)
- Rocket.rs (HTTP)
- PostgreSQL (DB)
